## Installation
Add this line to your application's Gemfile:

```ruby
gem 'hyku_plugin_google_cloud_url_mask'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install hyku_plugin_google_cloud_url_mask
```
